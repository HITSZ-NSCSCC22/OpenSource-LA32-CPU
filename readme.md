## 目录结构
```
    mycpu/  
           -> src  CPU内核，能够运行Linux。替换Chiplab的mycpu即可
           -> instruction  关于IP配置的说明
           -> ip  运用到的IP
           -> bit 比特流，可直接烧录
           
    peripheral/  LCD控制器代码，实现了触碰，显示，字符库功能。内含外设开发文档
 

```
